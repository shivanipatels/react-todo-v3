 # React Todo App - Version 3

This is **Version 3** of my Todo App built using **React** and **Context API**.  
It demonstrates centralized state management, clean component architecture, and modern React practices.

---

## Features
- Add todos with a due date
- Delete todos
- Centralized state using React Context API
- Form submit handling (Enter key supported)
- Uncontrolled inputs using `useRef`
- Component-based architecture:
  - `App.js`
  - `AddTodo`
  - `TodoList`
  - `TodoItem`
  - `WelcomeMessage`
- Conditional rendering for empty list message
- Styled with CSS Modules + Bootstrap 5

---

## Tech Stack
- React
- React Icons (`react-icons`)
- Bootstrap 5
- CSS Modules
